10月这波大片太炸了，每一部都吊打内娱


大王叫我来巡山，
巡往南山巡北山。


今年的国庆档，整体平庸。
除了动画片以外，五部新片都已在豆瓣开分。
均在6~7分的中庸水准，最高也只有7.3。
而且目测还有下降的趋势。

重头戏的国庆档，已然如此。
接下来的10月院线，更是没有多少看头。
对比之下，海外院线就让人眼馋了。
无论是好莱坞，还是日韩，都有重量级名导和演员回归。
今天，我就为大家整理了10月即将「环大陆上映」的新片。
到底有多少能在我们的大银幕看到呢？
《花月杀手》
导演：马丁·斯科塞斯
主演：莱昂纳多·迪卡普里奥 / 罗伯特·德尼罗 / 杰西·普莱蒙
上映日期：2023-10-19（中国香港）/2023-10-20（美国）

年度第一史诗大片，来了。
导演是81岁的马丁·斯科塞斯。
一把年纪的他，依然保持高能的创作热情，且看且珍惜。
主演则有小李子、罗伯特·德尼罗、杰西·普莱蒙、布兰登·费舍等一票大明星。
光是这个阵容，就没有任何理由拒绝。

和斯科塞斯上一部《爱尔兰人》相似，《花月杀手》片长也达到了200分钟+。
这「憋尿」的规格，已经透出了史诗的气息。
先前在戛纳首映，就收获不俗的评价。
豆瓣8.1，烂番茄新鲜度97%。
在不少影评人看来，这将是斯科塞斯生涯最佳作品之一。

一直以来，马丁·斯科塞斯都被誉为「电影社会学家」。
他的电影并不是简单的暴力犯罪片，而是借这些犯罪故事，剖析美国的社会问题和复杂人性。
这一次，他的视角从爱尔兰移民，转向了印第安原住民。
电影改编自同名小说，讲述历史上的一桩印第安人谋杀惨案。
该案件直接促使美国成立FBI，并逐渐发现更大的阴谋。
关于印第安人的题材，在今天的��莱坞已经很少见到。
不知斯科塞斯又会给我们带来怎样的惊喜。

《杀手》
导演：大卫·芬奇
主演：迈克尔·法斯宾德 / 蒂尔达·斯文顿 / 艾利斯·霍华德
上映日期：2023-10-27（美国）

大卫·芬奇一出手，影迷圈又要躁动了。
他是悬疑片大师，是电影语言的革新先驱，还是暗黑故事的重度迷恋者。
从早年的《七宗罪》《搏击俱乐部》，到后来的《社交网络》《龙纹身女孩》《消失的爱人》……都是影迷津津乐道的经典代表。
美剧方面，也有《纸牌屋》《心灵猎人》这种高分神作。
而动画片领域，他也作为制片人和分级导演，缔造了一部神作系列��爱死亡机器人》。

这一回，芬奇再次回归拍摄犯罪悬疑片。
卡司也是顶级配置。
由「法鲨」迈克尔·法斯宾德、蒂尔达·斯文顿主演。

电影讲述一个偏执、严谨的杀手，因为一次失手而展开一场追捕。
故事，一看就很带劲。
视听风格，也是一如既往的芬奇式快剪辑、细节控。

而「法鲨」也是许久不见了。
过去几年，他几乎没��接什么新片，而是作为赛车手参加勒芒系列赛。
甚至一度拿到亚军奖杯。
此次重回影坛，想必是大卫芬奇的邀约没法拒绝吧！

《止痛骗》
导演：大卫·叶茨
主演：艾米莉·布朗特 / 克里斯·埃文斯 / 凯瑟琳·欧哈拉
上映日期：2023-10-27（流媒体）

两位主演，相当养眼。
女神艾米莉·布朗特，和男神克里斯·埃文斯。
俊男靓女，上演诈骗计划。

电影是一场跟医药有关的骗局。
艾米莉·布朗特饰演的母亲，为了照顾好女儿，在一家初创医药公司找到工作。
凭借她出色的能力，她将这家惨淡经营的公司起死回生。
她看似走向了人生巅峰，实则卷入了一场阴谋之中。

对于国内观众来说，本片的导演大卫·叶茨也是老熟人。
曾拍过后四部《哈利·波特》电影，以及三部《神奇动物》系列电影。
他曾表示本片的风格和《华尔街之狼》相似。
后者极力展现纸醉金迷的奢靡生活，后又展现人性的丑陋与悲哀。
不得不说，从阵容到故事，都狠狠吊足胃口。

《华尔街之狼》
《自���职业者》
导演：皮埃尔·莫瑞尔
主演：约翰·塞纳 / 爱丽森·布里 / 胡安·巴勃罗·拉瓦
上映日期：2023-10-06（美国）

一部看着很简单无脑的动作爽片。
主演是「和平使者」约翰·塞纳。
而导演则是拍出过《飓风营救》《暴力街区》《玩命快递》(摄影指导)等经典动作大片的法国导演皮埃尔·莫瑞尔。
没太多花招，但每一次出击都简单有效，干脆利落。
这一次，能不能再让我们爽一回。

《蜘蛛网》
导演：金知云
主演：宋康昊 / 林秀晶 / 吴正世
上映日期：2023-09-27（韩国）/2023-10-13（中国台湾）

《蜘蛛网》很早就吊起了我的胃口。
导演是拍出过《看见恶魔》《死不张扬离奇失魂事件》的金知云。
论怪咖鬼才，韩国没有几个能比得了他。

《看见恶魔》
主演是宋康昊，大家熟悉的大饼脸。
演技实力无可指摘。

最有趣的是本片的题材。
故事聚焦1970年代的韩国电影审查制度。
为我们呈现一出混乱、荒谬、令人哭笑不得的戏中戏故事。
一点一滴，都深深戳着内地电影人和观众的痛点。
引进是没指望了，等着看网上的吧。
《1947波士顿》
导演：姜帝圭
主演：河正宇 / 任时完 / 裴晟祐
上映日期：2023-09-27（韩国）/2023-10-06（中国台湾）

河正宇，再次回归大银幕。
又加上任时完的搭配，让本片被寄予厚望。
更值得关注的，则是导演姜帝圭。
正是他在1999年拍出的《生死谍变》，开创了韩国电影新纪元，拉开了韩国电影崛起的序幕。
之后的《太极旗飘扬》，再一次创下票房奇迹，捅破了韩国本土电影票房天花板。

《太极旗飘扬》
这一回，他选择了体育传记题材。
聚焦二战后的第一场国际马拉松比赛，一个韩国运动员的挣扎与奋斗。

近年来，我们也拍过不少运动题材电影。
但无论是叙事角度还是拍摄手法，还是显得比较单一和局限。
相较之下，韩国电影虽也有宏大历史打底，但最关心的还是时代洪流下的个体命运。
挖掘出那些真正能让普通观众共情，真正拥有感染力的部分。
且看这次，姜帝圭和河正宇的组合，是否还会再创神迹。

《芭蕾复仇曲》
导演：李聪贤
主演：全钟瑞 / 金智勋 / 朴有林
上映日期：2023-10-06（韩国网络）

全钟瑞，韩国第一疯批女演员。
处女作《燃烧》，就大胆一脱。
凭借那段夕阳下神秘而唯美的独舞，刚出道就爆火。

第二部电影《电话》，饰演了一个反社会人格的恶女。
从最开始楚楚可怜的受害者，到嗜血的杀人魔，她只用了一个镜头。

这一次，她又将饰演一个疯批复仇者。
为了好友，一杀到底。
年度复仇大爽片，非她莫属。


《奇怪的生活方式》
导演：佩德罗·阿莫多瓦
主演：佩德罗·帕斯卡 / 伊桑·霍克 / 佩德罗·卡萨布兰科
上映日期：2023-10-04（美国）

第一眼看，好奇怪的片子。
阿莫多瓦居然拍佩德罗和伊桑·霍克的「兄弟情」。
太怪了！

不行，再看一眼！
越看越是满脸姨母笑。

作为阿莫多瓦久违的作品，虽然只是短片，却还是无比吸睛。
不光因为演员阵容，也因为这极具诱惑力的故事。
按照导演的说法，本片是他对《断背山》的致敬。
一位神枪手拜访25年的警长朋友，两人旧情复燃。

至于到底怎么「复燃」，我也是十分好奇。
幸好，这部短片将在10月20日上线流媒体。
到时候，我一定要第一时间「批判一下」。
《勿言推理 电影版》
导演：松山博昭
主演：菅田将晖 / 松下洸平 / 町田启太
上映日期：2023-10-12（中国香港）

去年，日剧《勿言推理》成了开年的一匹黑马。
看似是推理，其实是治愈。
话痨的主角通过言语来让身边人实现自我和解，给予心灵的治愈。
最终，在豆瓣上收获了8.4分的不俗评价。
没想到，今年直接来了一部电影版。
同样的爆炸头，同样的话痨萌男。
作为一个推理迷，我早早地期待住了。

《返朴》
导演：高畑秀太
主演：二宫和也 / 波瑠 / 桐谷健太
上映日期：2023-10-06（日本）

本片有两大看点。
其一，电影改编自北野武同名小说。
而且，这还是一部纯爱小说。
男主在咖啡厅遇上了神秘女子，瞬间因为价值观相同而被对方吸引。
自那之后，两人便相约每周四在咖啡厅相见。
男主感情不断加深，却似乎始终不明女子的心意。
难以预测的是，笑到最后的到底是不是「纯爱战神」。

这样的故事，颇有一些《花束般的恋爱》的味道。
只不过是「未完成版」。
更难让人想象的是，北野武这样一个老炮儿能写出怎样的纯爱故事。
单靠这样的反差，就让我对作品多了许多好奇。

其二，是神仙阵容。
导演高畑秀太，主演是二宫和也，再加上原作作者北野武。
这也是三人自2015年的《红鳉鱼》（豆瓣9.0）之后的又一次合作。
顶级阵容回归，不免让一众老粉也高呼一声「爷青回」。

《红鳉鱼》
《Kyrie之歌》
导演：岩井俊二
主演：AiNA THE END / 松村北斗 / 黑木华 / 广濑铃
上映日期：2023-10-13（日本）

本片的导演，岩井俊二。
恐怕是无数中国影迷的「初恋导演」了。
早年的《情书》《花与爱丽丝》，都是一代人的爱情圣经。

近些年，岩井俊二的创作略有下滑。
但，还是能带来不少惊喜。
这部《Kyrie之歌》，讲述一个流浪歌手的故事。
风格似乎很像回到早年《燕尾蝶》或《关于莉莉周的一切》那样的音乐电影。
主要则包括了松村北斗、黑木华、广濑铃等新锐演员，以及一位年轻的创作歌手。
这一次，岩井俊二还会带给我们怎样的浪漫。

以上，就是为大家盘点的本月值得关注的新片。
也许无缘在大银幕看到，但至少是个盼头。
让更多人知道，还有那些好片值得等待。
毕竟，坐在影院看电影是一种享受。
而等待电影放映的过程，则是将对一部电影的喜爱无限拉长。
